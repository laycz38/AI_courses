## 一、什么是神经网络

人工神经网络(ANN) = 模仿生物神经元的计算模型。

**核心公式**：
```
神经元的输出 = 激活函数(输入 × 权重 + 偏置)
              = f(w₁x₁ + w₂x₂ + ... + wnxn + b)
```

- **输入层** → 接收外部数据
- **隐藏层** → 内部特征变换
- **输出层** → 产生最终结果

> 关键认知：如果不加激活函数，不管堆多少层，整个网络等价于一个线性模型。激活函数注入非线性，让网络能拟合任意函数。

## 二、激活函数

| 函数 | 公式 | 输出范围 | 特点 | 使用场景 |
|------|------|---------|------|---------|
| **Sigmoid** | 1/(1+e⁻ˣ) | (0, 1) | 梯度消失严重（导数≤0.25），不以0为中心 | 仅用于二分类输出层 |
| **Tanh** | (eˣ-e⁻ˣ)/(eˣ+e⁻ˣ) | (-1, 1) | 以0为中心，仍会梯度消失 | 隐藏层可尝试 |
| **ReLU** | max(0, x) | [0, ∞) | 计算快，x>0不饱和，但有"神经元死亡"问题 | **隐藏层首选** |
| **Softmax** | eⁿ/Σeⁿ | (0,1)，和为1 | 将多个输出转为概率分布 | 多分类输出层 |

### 选择原则

| 位置 | 选什么 |
|------|--------|
| 隐藏层 | **ReLU** 默认，效果不好再试 LeakyReLU |
| 输出层-二分类 | Sigmoid |
| 输出层-多分类 | Softmax |
| 输出层-回归 | 不用激活函数（identity） |
| ⚠️ | 少用 Sigmoid，隐藏层尤其不要用 |

### 代码示例

```python
import torch
import torch.nn as nn

# 在 forward 中使用
x = torch.relu(x)       # ReLU
x = torch.sigmoid(x)    # Sigmoid
x = torch.tanh(x)       # Tanh
x = torch.softmax(x, dim=-1)  # Softmax
```

## 三、参数初始化

好的初始化让训练更快、更稳定。

### 常用方法

| 方法 | 说明 | 适用 |
|------|------|------|
| 均匀分布 | `nn.init.uniform_(w)`，默认(0,1) | 简单 |
| 正态分布 | `nn.init.normal_(w, mean=0, std=1)` | 简单 |
| 全0/全1 | `nn.init.zeros_(w)` / `nn.init.ones_(w)` | 偏置初始化 |
| 固定值 | `nn.init.constant_(w, 5)` | 特定场景 |
| **Xavier** | 均匀: [-√(6/(n_in+n_out)), +√(6/(n_in+n_out))] | **Sigmoid/Tanh** |
| **Kaiming (He)** | 正态: N(0, √(2/n_in)) | **ReLU** |

### 选择原则

```
激活函数 → 初始化方法
Sigmoid/Tanh → Xavier
ReLU/LeakyReLU → Kaiming/Normal
浅层网络 → 随机初始化就行
```

> PyTorch 每个层有默认初始化方法，一般够用。知道什么时候换就行。

## 四、神经网络搭建

### 基础模板

```python
import torch.nn as nn

class MyModel(nn.Module):
    def __init__(self):
        super().__init__()
        # 定义层
        self.linear1 = nn.Linear(20, 128)
        self.linear2 = nn.Linear(128, 256)
        self.out = nn.Linear(256, 4)

        # 初始化（可选）
        nn.init.kaiming_normal_(self.linear1.weight, nonlinearity='relu')
        nn.init.zeros_(self.linear1.bias)

    def forward(self, x):
        # 数据流
        x = torch.relu(self.linear1(x))
        x = torch.relu(self.linear2(x))
        x = self.out(x)          # 分类问题不要在forward里加softmax！
        return x
```

### 参数计算

```
全连接层参数量 = 输入维度 × 输出维度 + 输出维度(偏置)
```

例如 `nn.Linear(3, 3)` 有 3×3 + 3 = **12** 个参数。

## 五、损失函数

损失函数衡量"模型现在有多差"。

| 任务 | 损失函数 | PyTorch |
|------|---------|---------|
| 回归 | 均方误差 MSE | `nn.MSELoss()` |
| 二分类 | 二分类交叉熵 BCE | `nn.BCEWithLogitsLoss()` |
| 多分类 | 交叉熵 CrossEntropy | **`nn.CrossEntropyLoss()`** |

> `nn.CrossEntropyLoss()` 内部已包含 softmax，模型 forward 最后一层不要加 softmax！

```python
criterion = nn.CrossEntropyLoss()
loss = criterion(model_output, labels)   # labels 是整数类别索引
```

## 六、正则化

防止过拟合——训练集表现好，测试集差。

### Dropout（随机失活）

```python
dropout = nn.Dropout(p=0.4)
y = dropout(y)  # 40%的神经元输出变0，其余×1/(1-0.4)
```

- **训练时**生效：以概率 p 随机把神经元输出置 0
- **测试时**不生效：PyTorch 自动处理，不需要手动切
- 防过拟合最常用的手段之一

### Batch Normalization（批量归一化）

```python
bn = nn.BatchNorm1d(num_features)
x = bn(x)  # 标准化后做缩放+平移
```

- 对每层输出做标准化（均值→0，方差→1），再做可学习的缩放和位移
- 加速训练、允许更大的学习率、有一定正则化效果
- CV 领域使用更多

## 七、网络优化方法

PPT 中主要讲解了学习率调整策略：

| 策略 | 方式 |
|------|------|
| 等间隔衰减 | 每 N 个 epoch 学习率乘以 γ |
| 指定间隔衰减 | 在设定 epoch 节点调整 |
| 指数衰减 | 按指数函数连续衰减 |

SGD vs Adam：
- **SGD** — 基础优化器，需要仔细调学习率
- **Adam** — 自适应学习率，大多数情况更快收敛，推荐首选

## 八、案例：手机价格分类

### 问题

20 个手机特征 → 预测价格区间（0/1/2/3 四个档），多分类问题。

### 完整流程

```python
import torch, torch.nn as nn, torch.optim as optim
from torch.utils.data import TensorDataset, DataLoader
import pandas as pd, numpy as np
from sklearn.model_selection import train_test_split

# 1. 准备数据
data = pd.read_csv('data/手机价格预测.csv')
x = data.iloc[:, :-1].astype(np.float32)
y = data.iloc[:, -1].astype(np.int64)

x_train, x_valid, y_train, y_valid = train_test_split(
    x, y, train_size=0.8, random_state=88)

train_dataset = TensorDataset(torch.from_numpy(x_train.values),
                               torch.tensor(y_train.values))
valid_dataset = TensorDataset(torch.from_numpy(x_valid.values),
                               torch.tensor(y_valid.values))

# 2. 构建模型 (20 → 128 → 256 → 4)
class PhonePriceModel(nn.Module):
    def __init__(self, input_dim, output_dim):
        super().__init__()
        self.linear1 = nn.Linear(input_dim, 128)
        self.linear2 = nn.Linear(128, 256)
        self.linear3 = nn.Linear(256, output_dim)

    def forward(self, x):
        x = torch.relu(self.linear1(x))
        x = torch.relu(self.linear2(x))
        return self.linear3(x)  # 不加 softmax！

# 3. 训练
model = PhonePriceModel(input_dim=20, class_num=4)
criterion = nn.CrossEntropyLoss()
optimizer = optim.SGD(model.parameters(), lr=1e-3)

for epoch in range(50):
    for x, y in DataLoader(train_dataset, batch_size=8, shuffle=True):
        output = model(x)
        loss = criterion(output, y)
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()

# 4. 评估
correct = 0
for x, y in DataLoader(valid_dataset, batch_size=8):
    y_pred = model(x).argmax(dim=1)
    correct += (y_pred == y).sum()
print(f'Acc: {correct.item() / len(valid_dataset):.4f}')  # 0.6425

# 5. 保存模型
torch.save(model.state_dict(), 'model/phone.pth')
```

### 提升准确率的调优方向

- SGD → **Adam**
- 学习率 1e-3 → 1e-4
- **对数据标准化**
- 增加网络深度/宽度
- 增加训练轮次
- 加 Dropout / BN

## 九、神经网络的优缺点

| 优点 | 缺点 |
|------|------|
| 精度高，可逼近任意非线性函数 | 黑箱，难以解释 |
| 丰富的框架和库支持 | 训练时间长，需要大量算力 |
| 在某些领域超过人类 | 需要调整超参数 |
| | 小数据集表现差，容易过拟合 |
