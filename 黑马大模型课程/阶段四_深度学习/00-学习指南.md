### 第三步：神经网络三板斧，记住这个公式

每个神经网络本质上就是：

```
输入 → 线性层(权重×输入 + 偏置) → 激活函数(加非线性) → 输出
```

PPT 02 的框架就是：

1. **搭骨架** — `nn.Linear(in, out)` 定义每一层
2. **加非线性** — 隐藏层用 ReLU，输出层按任务选（二分类 sigmoid / 多分类 softmax / 回归不用）
3. **算损失** — 分类用 `CrossEntropyLoss`，回归用 `MSELoss`
4. **更新参数** — `loss.backward()` 算梯度 → `optimizer.step()` 更新权重

上面这四步，就是所有深度学习代码的模板。手机价格分类案例 = 这个模板套一遍。

### 第四步：重点理解，别跳过的

| 概念 | 一句话 | 为什么重要 |
|------|--------|-----------|
| 梯度下降 | 沿着让损失变小的方向，一步步调整参数 | 所有模型训练的本质 |
| 反向传播 | 自动算每个参数对损失的贡献（梯度） | 不用手算导数，PyTorch 帮你做 |
| 激活函数 | 给网络加非线性，否则不管多少层都等于一层 | ReLU 是现在的默认选择 |
| 过拟合 + Dropout | 训练集上表现好、测试集差 = 过拟合；随机关掉一些神经元来防这个 | 实际训练必用 |
| 损失函数 | 衡量"模型现在有多烂"的数字 | 训练的唯一目标是让这个数字变小 |

### 第五步：动手写的三个案例，顺序别乱

1. **线性回归**（PPT 01 末尾）— 最简单的，只有一层 Linear，理解训练循环
2. **手机价格分类**（PPT 02）— 三层 Linear + ReLU + CrossEntropyLoss，理解分类问题
3. **自己改一个** — 把手机数据集换成年收入/房价预测，或者把层数从 3 层改成 5 层，看看有什么变化

## 常见坑

1. **张量形状不匹配** — 矩阵乘法要求 `(n,m) @ (m,p)`，报错就打印 `.shape` 查
2. **忘记 `zero_grad()`** — 梯度会累加，每个 batch 开始前要清零
3. **`requires_grad=True` 的张量不能直接转 numpy** — 先 `.detach()` 再 `.numpy()`
4. **`nn.CrossEntropyLoss` 自带 softmax** — 模型最后一层不要手动加 softmax
5. **数据没转 float32** — 模型参数默认 float32，输入得匹配，否则报 dtype 错误
