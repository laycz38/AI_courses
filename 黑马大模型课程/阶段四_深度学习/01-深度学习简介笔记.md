## 一、什么是深度学习

### 层级关系

```
人工智能 (AI)
  └── 机器学习 (ML)        ← 实现 AI 的一种途径
        └── 深度学习 (DL)   ← 实现 ML 的一种方法
```

> 深度学习 = 基于**多层人工神经网络**，每层对输入做非线性变换，逐层提取更抽象、更复杂的特征。

### 与传统机器学习的区别

| | 传统机器学习 | 深度学习 |
|------|------------|---------|
| 特征提取 | 人工设计 | 算法**自动**提取 |
| 擅长数据 | 结构化数据 | 高维数据（图像、语音、文本） |
| 可解释性 | 相对好 | 差（黑箱） |
| 算力需求 | 低 | 高（需要 GPU） |

---

## 二、深度学习四大特点

1. **多层非线性变换** — 低层捕捉简单特征（边缘、颜色），高层识别复杂模式（物体、面部）
2. **自动特征提取** — 不需要人工特征工程，从原始数据中端到端学习
3. **大数据+GPU算力** — 数据量和算力是深度学习效果的关键
4. **可解释性差** — "黑箱"问题，难以解释为什么做出某个决策

---

## 三、常见深度学习模型

| 模型 | 擅长的领域 |
|------|-----------|
| **CNN** 卷积神经网络 | 图像识别、计算机视觉 |
| **RNN** 循环神经网络 | 序列数据、时间序列 |
| **Autoencoders** 自编码器 | 数据压缩、降噪、特征学习 |
| **GAN** 生成对抗网络 | 图像生成、数据增强 |
| **Transformer** | NLP、大模型基础架构 |
| **DRL** 深度强化学习 | 游戏、机器人控制 |
| **GNN** 图神经网络 | 社交网络、分子结构 |

## 五、发展历史

### 人工智能三次浪潮

| 时期 | 流派 | 标志事件 |
|------|------|---------|
| 1950s-70s | **符号主义** | 图灵设计象棋程序(1950)，IBM跳棋战胜人类(1962) |
| 1980s-2000 | **统计主义** | Vapnik的SVM(1993)，IBM深蓝战胜卡斯帕罗夫(1997) |
| 2012-至今 | **神经网络/深度学习** | AlexNet(2012)，AlphaGO战胜李世石(2016) |
| 2017-至今 | **大规模预训练模型** | Transformer(2017) → Bert/GPT(2018) → ChatGPT(2022) |

### 深度学习关键节点

| 年份 | 事件 |
|------|------|
| 1940s | McCulloch-Pitts 神经元模型 |
| 1957 | Rosenblatt 感知器 |
| 1960s | 多层感知器 MLP（受限计算能力） |
| 1986 | **反向传播算法(BP)** — Rumelhart 等人发表 |
| 2006 | Hinton 提出深度置信网络(DBN)，**深度学习复兴** |
| 2012 | AlexNet 在 ImageNet 取得巨大成功，**爆发期开始** |
| 2016 | AlphaGO 战胜李世石 |
| 2017 | Transformer 框架出现 |
| 2018 | BERT、GPT 出现 |
| 2022 | ChatGPT 出现，进入大模型 AIGC 时代 |
